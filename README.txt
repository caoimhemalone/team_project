You MUST be connected to the internet for database reading/writing to work, otherwise it will crash and you won't be able to access the app.

When signing up, student/staff emails will not work, use a hotmail or gmail etc.

There is no setup for database, it is all done in the code.

If running on Android Studio make sure to gradle sync the project before running.

If running on an android device you can find the app-debug.apk file by navigating to app/build/outputs/apk

All code is on GitHub at: https://github.com/caoimhemalone/team_project
